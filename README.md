#|==================|IdentiChip|================|

Chipset and Device information Gathering for USB Wireless and Bluetooth Adapters.
These tools and commands allow a user to find hardware specific information on chipset and device
hardware information on unnamed or unknown USB Wireless and Bluetooth adapters.
With this information in hand, a user can possibly gain low-level access to device and or firmware.
Also assist in sourcing correct drivers for the USB Wireless and Bluetooth adapters.

This project is still in very early stages of development. 
Repository for Tools and scripts still to be update with a .pkg build
